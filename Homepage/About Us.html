About Us



Situs Web Tentang Kopi



Jenis website yang akan kami buat adalah website bidang ilmu kopi, dimana kami akan menjelaskan sejarah/asal-usul kopi serta berbagai macam cara menyeduh kopi agar semakin diminati oleh banyak orang. Website yang akan kita buat juga akan dibuat menarik sehingga banyak orang yang tertarik.



Visi dan Misi

Visi

Menunjukkan bahwa ada banyak pengetahuan tentang kopi yang bisa membuat kopi lebih diminati dan dinikmati dengan berbagai cara menyeduh, mencampur bahan, dll. Lalu Dengan platform situs web ini kami berharap dapat membuat lebih banyak orang tertarik dengan kopi.



    Misi :

  1. Memberikan berbagai macam ilmu tentang kopi
  2. Menjaga cita rasa kopi yang legendaris dari generasi ke generasi.
  3. Meningkatkan kreativitas untuk membuat menu baru yang lebih bervariasi.

The Software We Use

Visual Studio Code

Visual Studio Code adalah editor kode sumber gratis yang dibuat oleh Microsoft untuk Windows, Linux, dan macOS. Fitur termasuk dukungan untuk debugging, syntax highlighting, intelligent code complete, snippet, code refactoring, dan embedded Git.

Github Desktop

GitHub, Inc. adalah perusahaan multinasional Amerika yang menyediakan hosting untuk pengembangan perangkat lunak dan kontrol versi menggunakan Git. Ia menawarkan kontrol versi terdistribusi dan fungsi manajemen kode sumber (SCM) dari Git, ditambah fitur-fiturnya sendiri.

Notepad++

Notepad ++ adalah editor teks dan kode sumber untuk digunakan dengan Microsoft Windows. Ini mendukung pengeditan tab, yang memungkinkan bekerja dengan beberapa file terbuka dalam satu jendela.
