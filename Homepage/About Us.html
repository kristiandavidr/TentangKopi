About Us



A Website About Coffee



The type of website that we will create is a website in the field of coffee science, where we will explain the various ways of brewing coffee so that it can be increasingly attractive to many people. The website that we will create will also be made attractive so that many people will be interested



Vision and Mission

Vision

Tell and show that there is a lot of knowledge about coffee that can make coffee more desirable and enjoyable by various ways of brewing, mixing ingredients, etc. Then with this website platform we hope to make more people interested in coffee.



    Mission :

  1. Provides various kinds of knowledge about coffee
  2. Maintain the legendary taste of coffee from generation to generation.
  3. Increase creativity to create new, more varied menus.

GitHub, Inc. is an American multinational corporation that provides hosting for software development and version control using Git. It offers the distributed version control and source code management (SCM) functionality of Git, plus its own features.

Notepad++ is a text and source code editor for use with Microsoft Windows. It supports tabbed editing, which allows working with multiple open files in a single window.
